# FleetCore 

## Car Rental & Fleet Management API

FleetCore is a backend REST API for managing vehicle rentals, bookings, and fleet operations. The project is being developed using Java, Spring Boot, Spring Security, and PostgreSQL, following a commit-by-commit development approach.

 Project Status: In Progress

### Planned Features
- JWT-based authentication and authorization
- Role-based access control (Admin / Customer)
- Car fleet management (CRUD operations)
- Booking management with date-range conflict validation
- Payment workflow for booking confirmation
- Exception handling and API validation
- RESTful API documentation

### Tech Stack
- Java 17
- Spring Boot 3
- Spring Security + JWT
- PostgreSQL
- Spring Data JPA (Hibernate)
- Maven
